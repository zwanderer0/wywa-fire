# Deploy WYWA blog updates
1. Pull the latest changes that add the Astro blog, local image assets, and build scripts.
2. Push commit e9ba3c0 (message: “Add Astro blog and local assets for updates”) to the GitHub repo zwanderer0/wywa-fire:
   git push origin main
3. Ensure the deployment pipeline (Netlify or equivalent) runs npm run build so both the React SPA and the new /updates Astro output land in production.
4. Verify production paths:
   - https://wywa.ai/
   - https://wywa.ai/updates/
   - https://wywa.ai/updates/2025-01-31-command-terminal-refresh/
5. Update Decap CMS OAuth settings (updates/public/admin/config.yml) with the correct base_url/auth_endpoint/app_id if not already done, so only authorized GitHub users can log in at /updates/admin/.
